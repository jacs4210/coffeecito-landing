# Iniciar servidor local

Usa este workflow para previsualizar los cambios de diseño en tu entorno de desarrollo local.

1. Asegúrate de encontrarte en el directorio raíz del proyecto (`10_landing_coffeecito`).
2. Levanta un servidor local en el puerto `3000` con `npx serve`.
// turbo
3. Ejecutar comando de servidor:
npx serve -s . -p 3000
